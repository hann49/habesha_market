# Habesha Market — Backend

E-commerce REST API built with NestJS, TypeORM, and PostgreSQL.

## Tech Stack
- NestJS (Node.js + TypeScript)
- PostgreSQL
- TypeORM
- JWT Authentication

## Modules
- auth — Register, Login, JWT
- users — User management
- products — Product CRUD
- cart — Shopping cart
- orders — Order placement
- sellers — Seller dashboard
- admin — Admin controls

## Setup
1. Clone the repo
2. Run npm install
3. Create .env file (see .env.example)
4. Run npm run start:dev

## API runs on
http://localhost:3000
