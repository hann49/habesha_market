# Habesha Market — User Stories

**Format:** "As a [role], I want to [action], so that [benefit]."

---

## 👤 Customer Stories

1. As a **customer**, I want to **register an account**, so that **I can shop on the platform**.
2. As a **customer**, I want to **log in securely**, so that **my account stays protected**.
3. As a **customer**, I want to **browse all products on the homepage**, so that **I can discover what's available**.
4. As a **customer**, I want to **search for products by name**, so that **I can find items quickly**.
5. As a **customer**, I want to **filter products by category and price**, so that **I can narrow down my choices**.
6. As a **customer**, I want to **view detailed information about a product**, so that **I can decide whether to buy it**.
7. As a **customer**, I want to **add products to my cart**, so that **I can buy multiple items together**.
8. As a **customer**, I want to **update quantity or remove items from my cart**, so that **I can adjust my order**.
9. As a **customer**, I want to **checkout and place an order**, so that **I can complete my purchase**.
10. As a **customer**, I want to **view my order history**, so that **I can track my past purchases**.
11. As a **customer**, I want to **see the status of my order (pending, shipped, delivered)**, so that **I know when to expect delivery**.

---

## 🛍️ Seller Stories

12. As a **seller**, I want to **create a seller account**, so that **I can sell on the platform**.
13. As a **seller**, I want to **add new products with images and details**, so that **customers can buy them**.
14. As a **seller**, I want to **update my product information**, so that **the listing stays accurate**.
15. As a **seller**, I want to **delete products I no longer sell**, so that **my shop stays clean**.
16. As a **seller**, I want to **manage my product stock**, so that **I don't oversell items**.
17. As a **seller**, I want to **view orders for my products**, so that **I can prepare them for shipping**.

---

## 👨‍💼 Admin Stories

18. As an **admin**, I want to **view all registered users**, so that **I can monitor the platform**.
19. As an **admin**, I want to **block or delete suspicious users**, so that **the platform stays safe**.
20. As an **admin**, I want to **remove inappropriate products**, so that **the marketplace stays trustworthy**.
21. As an **admin**, I want to **view platform statistics (users, orders, revenue)**, so that **I can track growth**.
