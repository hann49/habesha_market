# Habesha Market — Frontend

E-commerce web app built with Next.js and Tailwind CSS.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Axios
- Zustand
- React Hook Form

## Pages
- / → Home page with hero section
- /login → Login
- /register → Register
- /products → Product listing with search and filter
- /products/[id] → Product details

## Setup
1. Clone the repo
2. Run npm install
3. Run npm run dev

## App runs on
http://localhost:3001
